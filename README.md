### Sobre la aplicación

Trading Journal - GlobalCode 2.0 es el diario de trading profesional más completo y visual para traders de Forex, Oro, Crypto, Índices y desafíos prop.

**Características principales:**
- Calendario inteligente con código de colores
- Monitor avanzado de Drawdown
- Calculadora de riesgo en vivo
- Estadísticas profesionales
- Backup completo + PDF reports
- Experiencia 100% offline y sin publicidad
